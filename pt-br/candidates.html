Candidatas

Fontes aguardando revisão.

Todo envio pelo formulário de issue chega aqui antes de fazer parte do catálogo público. "Em revisão" significa que passou pela validação automática e tem um pull request rascunho aberto. "Precisa de ajuste" significa que a validação encontrou um problema e ainda não existe um pull request — geralmente um campo obrigatório em branco ou uma opção que não correspondeu exatamente.

Nenhuma fonte está aguardando revisão no momento. Proponha uma ou abra diretamente uma nova submissão de fonte (abre em nova aba).

Esta lista se atualiza automaticamente conforme chegam envios, e pelo menos a cada seis horas de qualquer forma. Ela espelha os dados públicos de issues e pull requests deste repositório no GitHub — nada aqui é enviado ou hospedado pela AmazoniaDB.

A AmazoniaDB é um índice independente, não uma hospedeira de dados. Nomes e materiais das fontes permanecem com seus respectivos provedores.
