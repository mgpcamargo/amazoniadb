Support AmazoniaDB

Keep this directory independent.

AmazoniaDB doesn't run ads, sell data, or take paid placement — every entry is here because it earned it, not because someone paid for it. That only stays true if the maintenance work behind it is funded some other way.

What support actually covers

  • Reviewing every community submission by hand before it's merged — the "editorially reviewed" tier means someone actually checked it.
  • The weekly automated check that catches dead links and entries that haven't been re-verified in 180+ days.
  • Hosting and keeping the submission pipeline running.
  • Time spent expanding coverage into new domains and countries as the directory grows.
